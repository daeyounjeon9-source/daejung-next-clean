대중NEXT v02.1 - UI 기본 골격

실행 방법
1) 압축을 daejungnext_v02 폴더로 풉니다.
2) 터미널에서 daejungnext_v02 폴더로 이동합니다.
3) python main.py 를 실행합니다.

이번 버전 포함 내용
- main.py 실행 진입점
- 상단 상태바
- 왼쪽 1~5페이지 메뉴
- 중앙 페이지 전환 구조
- 하단 공통 로그바
- 1 메인 / 2 입력센터 / 3 실행센터 / 4 결과분석 / 5 시스템관리 화면 초안
- core 4개 파일 연결
- services 3개 파일 기본 연결

주의
- 실제 거래소 주문 기능은 아직 연결하지 않았습니다.
- 이번 단계는 화면 전환과 내부 연결 확인용입니다.
