DAEJUNGNEXT install/run tools

1. Copy these files into:
   Desktop\DAEJUNGNEXT_V02\daejungnext_v02

2. Run install.bat once.
3. Run run.bat whenever you want to start the program.

Alternative terminal commands:
cd %USERPROFILE%\Desktop\DAEJUNGNEXT_V02\daejungnext_v02
pip install -r requirements.txt
python main.py
