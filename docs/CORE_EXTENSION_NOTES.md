# DAEJUNG NEXT CORE 확장 작업 노트

이 버전은 기존 DAEJUNGNEXT_V02 프로젝트를 유지하면서 본사 운영센터 화면을 확장한 버전입니다.

## 반영 내용
- Dashboard 관제센터 고도화
- Sidebar 메뉴 확장
- 카드 클릭 → 전문 업무 화면 이동 구조
- 회원 관리 화면
- 주문 관리 화면
- 결제/정산 관리 화면
- 고객지원 센터 화면
- 안내 서비스 / CMS 화면
- 로그인 / 인증 관리 화면
- 플랫폼 운영 화면
- 통계 / 분석 화면
- 시스템 모니터링 화면
- 설정 / 권한 화면
- 로그인 화면 디자인 개선

## 실행
```bash
npm install
node server/index.js
```
브라우저에서 `http://localhost:1573/login.html` 접속

테스트 계정: `admin / 1234`
